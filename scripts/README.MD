# Running Scripts

## Running reviews scripts
1. Create a Python Virtual Environment
    ```
    python -m venv venv
    ```
2. Activate the virtual environment
    ```
    souce venv/Activate/scripts
    ```
3. Install the dependencies
    ```
    pip install requirements.txt
    ```
4. Go to `scripts/reviews/`
5. Run `python course_review.py`
    - For any help regarding usage, run `python course_review.py --help`
